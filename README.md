# 英文測驗官網

## 描述

> 網站是建立於2020年，為因應教師對小朋友進行英文測驗所使用。
> 其結構可分為html、css、javascript、ajax 與 googlesheet 所製作。

## 功能

### 流程

1. 小朋友填寫個人資料(學號)，並進行聽力測驗
1. 填答完畢，經由自動對答
1. 連結至 Google Sheet 表單給教師登記成績

### 特色
* 發音功能
* 倒數時間
* 透過ajax傳送
* 自動對答
* 自動keyin Google Sheet 中
